Beauty & Wellness Appointment Booking System

A responsive salon appointment booking website developed using HTML, CSS, JavaScript, and Firebase.
It allows users to conveniently book appointments for beauty and wellness services such as Facial, Hair Massage, Waxing, Spa, and Hydrafacial.
All booking details are securely stored in Firebase Firestore, and users receive instant confirmation messages after booking.
