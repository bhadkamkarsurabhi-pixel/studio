# **App Name**: Charmony E-Boutique

## Core Features:

- Product Catalog: Display and manage product listings with images, descriptions, prices, and categories.
- Shopping Cart: Allow users to add products to a cart, update quantities, and proceed to checkout.
- Secure Checkout: Integrate Stripe for secure payment processing, including test and live mode support.
- User Authentication: Implement user signup, login, and session management using Firebase Auth or JWT.
- Admin Dashboard: Provide an admin interface for managing products, orders, users, and discounts.
- AI Product Description Tool: Allow the user to generate and refine the description of the product. The tool uses the prompt from the store manager. The tool may also use information mined about similar product offerings that it searches for.
- Search Functionality: Enable users to search for products by name, category, or keywords, including a search bar in the navbar.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke luxury and tranquility.
- Background color: Cream (#F5F5DC), a lightly desaturated tone of lavender, providing a soft, elegant backdrop.
- Accent color: Gold (#FFD700) for highlights, buttons, and CTAs to symbolize premium quality; use sparingly.
- Headline font: 'Belleza' (sans-serif) for an elegant and fashionable feel.
- Body font: 'Alegreya' (serif) for readability and a contemporary, intellectual touch.
- Use minimalist line icons for categories, cart, user profile, and search.
- Employ a grid layout for product listings and a clean, spacious design overall.
- Subtle animations on product hover, cart updates, and page transitions to enhance user experience.
